You are to write the following Handlebars helpers.

  1. Write a helper named copyright. The helper should take one argument: the name of the copyright holder. The helper should result in the user seeing "©year name," where year is the current year and name is the value of the parameter.
    ?
    <p>Here's a deep thought: do caterpillars use toothbrushes?</p>
    {{copyright "Gabriel Foust"}}

    Here's a deep thought: do caterpillars use toothbrushes?

    ©2017 Gabriel Foust
    • You should display the copyright symbol using the &copy; HTML entity.
    • You should get the current year by creating a new JavaScript date object (if you don't specify a date when you construct a date object it defaults to the current date), then using its getFullYear method.
      ?
      var d = new Date();
      var year = d.getFullYear();
  2. Write a helper named codeblock. The helper should print the contents of the block expression in a <pre class="codeblock"> tag and break it into tokens. Each token should be wrapped in a <span class="className"> tag with the appropriate class name.
    ?
    <p>Here's some JavaScript code</p>
     
    {{#codeblock}}
    // Import express library
    var express = require('express');
     
    // Create express application
    var app = express();
    {{/codeblock}}
    The above template should render as shown below. (You may view the page source to see the HTML created for this example.)

    Here's some JavaScript code

    // Import express library
var express = require('express');

// Create express application
var app = express();
    • Use my tokens.js file as a reference for breaking the string into tokens.
    • Modify the loop so that when a token is found, instead of printing it to the console you wrap it in a span tag and concatenate it to the result.
    • The token styles must come from a style sheet. The codeblock helper is not responsible for this stylesheet; all the helper does is make sure the tokens have the appropriate class name.
